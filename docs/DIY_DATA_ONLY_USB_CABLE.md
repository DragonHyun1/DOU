# Data-Only USB 케이블 제작 가이드

## 🎯 목적
USB VBUS(5V) 충전 라인을 차단하여, ADB는 사용하면서 충전 간섭을 제거

## ⚡ 방법 1: 빠른 해결 - VBUS 핀 테이핑 (5분)

### 필요한 것
- 일반 USB 케이블 (현재 사용 중인 것)
- 절연 테이프 (또는 카톤 테이프)
- 가위

### 작업 순서

**Option A: USB-A (PC 쪽) 커넥터 수정**
```
USB-A 커넥터 (PC에 꽂는 쪽)
┌─────────────────┐
│  1  2  3  4     │  ← 핀 번호
│ ═══ ─── ─── ═══ │
└─────────────────┘
  ↑           ↑
Pin 1        Pin 4
(VBUS)       (GND)

Pin 1: VBUS (5V)   ← 이것을 막아야 함!
Pin 2: D- (Data)   ← ADB 사용
Pin 3: D+ (Data)   ← ADB 사용
Pin 4: GND         ← 필요
```

**절연 테이프 붙이기:**
```
1. 작은 테이프 조각을 준비 (2mm x 5mm 정도)
2. USB-A 커넥터의 Pin 1 (맨 왼쪽) 위에 붙이기
3. 두께가 얇으면 2-3겹 겹쳐 붙이기
4. PC USB 포트에 꽂아서 들어가는지 확인
5. 너무 두꺼우면 조금 자르기
```

**사진 가이드:**
```
Before:
┌─────────────────┐
│ ▓▓▓ ─── ─── ▓▓▓ │  ← 모든 핀 노출
└─────────────────┘

After:
┌─────────────────┐
│ [T] ─── ─── ▓▓▓ │  ← Pin 1에 테이프 [T]
└─────────────────┘
```

### 테스트 방법

```bash
# 1. 테이프 붙인 케이블로 연결
# 2. ADB 작동 확인
adb devices

# 3. 충전 상태 확인
adb shell dumpsys battery | grep "powered"
# 예상 출력:
#   AC powered: false
#   USB powered: false  ← 이것이 false여야 함!

# 4. DAQ로 Battery Rail 전압 측정
# 예상: 3.95V (4.15V가 아님!)
```

## 🔧 방법 2: 케이블 내부 VBUS 선 절단 (영구적)

### 필요한 것
- 일반 USB 케이블
- 커터칼 또는 니퍼
- 멀티미터 (옵션)
- 절연 테이프

### 작업 순서

1. **케이블 피복 벗기기**
   ```
   USB 케이블 중간 부분 5cm 정도 피복 제거
   ```

2. **내부 선 확인**
   ```
   표준 USB 케이블 내부:
   - 빨간색: VBUS (+5V)    ← 이것을 절단!
   - 흰색:   D- (Data)     ← 유지
   - 초록색: D+ (Data)     ← 유지  
   - 검은색: GND (Ground)  ← 유지
   
   주의: 케이블마다 색상이 다를 수 있음!
   ```

3. **VBUS 선 절단**
   ```
   빨간색 선만 1cm 정도 절단
   (양쪽 끝이 닿지 않도록!)
   
   ───[빨간선]─── → ───[빨간선]  [빨간선]───
                           ↑
                        1cm 간격
   ```

4. **절단 부위 절연**
   ```
   각각의 빨간선 끝을 절연 테이프로 감싸기
   (합선 방지)
   ```

5. **케이블 피복 재조립**
   ```
   전체를 절연 테이프로 감싸서 마무리
   ```

### 멀티미터로 검증 (옵션)

```
PC 쪽 USB-A 커넥터:
  Pin 1 (VBUS) ←→ 폰 쪽 Pin 1 (VBUS)
  
멀티미터 저항 모드로 측정:
  Before 절단: 0Ω (연결됨)
  After 절단:  ∞Ω (끊어짐) ✓
  
D+, D-, GND는 여전히 0Ω이어야 함!
```

## ⚠️ 주의사항

### 테이핑 방법
- ✅ 두께 조절이 가능 (여러 겹)
- ✅ 실패해도 테이프만 제거하면 원상복구
- ⚠️ 테이프가 떨어질 수 있음 (주기적 확인)

### 절단 방법  
- ✅ 영구적이고 확실함
- ✅ 테이프 떨어질 걱정 없음
- ❌ 실패하면 케이블 못 씀 (연습 필요)
- ❌ 일반 충전용으로 사용 불가

## 🎯 권장 순서

### 1단계: 즉시 테스트 (테이핑)
```
1. 기존 USB 케이블 Pin 1에 테이프 붙이기
2. ADB 연결 확인: adb devices
3. 충전 상태 확인: adb shell dumpsys battery
4. DAQ 전압 측정: 3.95V 확인
5. Phone App 시나리오 실행
```

**5-10분이면 즉시 테스트 가능!**

### 2단계: 안정적 사용 (선택)
```
- 테이핑이 잘 작동하면 계속 사용
- 또는 제품 구매
- 또는 케이블 절단 방식으로 영구 제작
```
